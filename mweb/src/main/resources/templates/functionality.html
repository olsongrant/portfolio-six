Virtual Portfolio Functionality

As of the end of January, 2021, the functionality in the Virtual Portfolio application is as follows:


Login and Registration

  • Traditional registration email username and registered password is offered
  • Passwords are kept in an encrypted format
  • Social media login with Facebook is supported only in development and test modes, as the application review with Facebook has yet to be done
  • each user can delete their own user record and erase their activities in the application

Account Management

  • a user can create one or more accounts
  • each account is initially funded with $100,000
  • users can engage in self-directed transactions of a selection of ten exchange-traded funds (ETFs)
  • users can browse and review accounts of other users
  • users can view the assets in accounts, including both the cash and the market value of the ETF assets in the account
  • the user can delete their own account(s)

Transactions and Asset Prices

  • prices of the ETF shares are retrieved from bigcharts.marketwatch.com. Prices are delayed (not real-time)
  • prices are retrieved from the bigcharts source every three minutes and cached within the system to help ensure fast page-loads
