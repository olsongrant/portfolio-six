About Formula Fund Virtual Portfolio

Virtual Portfolio can be thought of as an investing or trading game. It has the functionality of allowing a virtual investment portfolio to be established with a set of exchange-traded funds generally available to investors in the United States of America. The following functionality is available to everyone:

  • browsing the accounts currently in the system
  • viewing the quantities and values of the assets within each account

Additionally, those who register can enjoy the following functionality:

  • creating a user with a handle / nickname
  • opening one or more accounts
  • making purchases and sales within each account
  • deleting their own accounts as well as their user record

Registration and login can be done via (1) an email username and a password kept in encrypted form in the application database, or (2) by social media account (Google) authentication. For the password-based method, verification of the email address is required. A privacy policy statement is available.


The application has been developed with a strategy of focusing on a minimum viable product. Certainly, the scope is limited and I hope to add functionality in the future.


A primary motivation for building Virtual Portfolio has been to develop and showcase proficiency with specific software development frameworks, libraries, and techniques. A description of the application structure is available.
