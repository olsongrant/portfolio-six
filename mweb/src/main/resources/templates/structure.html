Structure and Techniques Used in the Virtual Portfolio application

This version of the Virtual Portfolio application has been built with the following technology:

  • Java 11
  • Spring Boot
  • Spring Web MVC
  • Thymeleaf templating engine
  • Spring Security 5
  • Spring Data JPA
  • Bootstrap CSS
  • Bootstrap Validator
  • MySQL

Deployment has been done on Amazon Web Services, specifically using the following AWS services:

  • Elastic Beanstalk
  • Route 53
  • EC2
  • Certificate Manager
  • Simple Email Service
  • RDS
