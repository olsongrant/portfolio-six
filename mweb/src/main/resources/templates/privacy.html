Privacy Policy

Virtual Portfolio

Virtual Portfolio is a stock market investing simulation game, offered for educational and entertainment value. In order to be an active participant, a would-be player must register as a user. There are two mechanisms offered for registration and login:

  1. Application-specific credentials stored in the application database, with the password stored in encrypted form
  2. Social media (Facebook) authentication

Information collected from users in the password-based registration process

When the user registers without the social media account, the following items are required:

  • First name
  • Last name
  • Email address
  • password
  • a handle (nickname)

The password provided to the application is stored in an encrypted form. Users are encouraged to utilize a unique password for the application. The application does have password reset functionality that utilizes an email message sent to the address used for the registration. The handle is the primary way that users are distinguished in the application.


Information collected from users who choose the social media account option

From the social media account, Virtual Portfolio collects the following:

  • First name
  • Last name
  • Email address
  • an arbitrary identifier provided by the social media platform

From the user, Virtual Portfolio requires a handle.


How are email addresses used by the application?
The email address is used only for communication directly associated with the application, such as the following:

  • confirmation of the address
  • password reset request authentication
  • update messages pertaining to the Virtual Portfolio application

The email address provided by the user will not be shared with third parties.


Any updates to the privacy policy will be preceded by a required acknowledgment at application login, so that users can decide whether to continue or not, prior to the application of the change.
